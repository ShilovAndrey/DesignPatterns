
В кофейне продают несколько видов кофе(HouseBlend, DarkRoast, Espresso, Decaf) и к каждому виду кофе можно добавить дополнительный компонент(Milk, Chocolate, Soy, Whip). Кроме того, порции могут быть разных размеров (LITTLE, MEDIUM, BIG).
Все напитки происходят от абстрактного класса Beverage.
Бармен должен името возможность внести основной напиток, получить по нему цену, добавить в основной напиток один/несколько компонентов.

Реализовать задачу, использую паттерн Декоратор.
