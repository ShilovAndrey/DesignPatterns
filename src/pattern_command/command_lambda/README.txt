Пульт имеет 7 программируемых ячеек (каждая из которых связывается с отдельным устройством) и
соответствующую кнопку "вкл/выкл" для каждой ячейки.

Необходимо использовать классы для управления домашними устройствами: светильники, вентиляторами,
акустическим оборудованием, ванна-джакузи.
Создать API для программирования пульта, чтобы каждая ячейка могла быть настроена на управление устройством или группой
устройств. Также следует учесть, что пульт должен поддерживать, как текущий набор устройств, так и все устройства,
которые могут быть добавлены в будущем.
Класс вентилятор должен иметь возможность включать устройство на разных скоростях при нажатии команд на разных ячейках.

Для упрощения кода использовать лямбда-выражения.
