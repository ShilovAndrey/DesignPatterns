
Дать определение паттерну Стратегия
Решить задачу, использую паттерн Стратегия:
Ниже описаны классы игровых персонажей и разных типов вооружения. Каждый персонаж в люой момент времени использует только один вид оружия, но может свободно менять оружие в ходе игры:

- есть класс Character.
- классы Queen, King, Knight, Troll
- виды оружия Knife,BowAndArrows,Axe,Sword
