
Есть метеоролигиский датчик(WeatherData), который отслеживает текущие погдные условия (температуру и давление).
Необходимо создать приложение, которое изначально отображает 2 визуальных элемента: текущую сводку и статистику.
Все данные обновляются в реальном времени, по мере того как объект WeatherData получает данные последних измерений.
Кроме того, необходимо предусмотреть возможность расширения программы. Определите API, что другие разработчики могли
писать собственные визуальные элементы для отображения погодной информации и поключать их к приложению.
